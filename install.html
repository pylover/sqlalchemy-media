Installing¶

PyPi¶

$ pip install sqlalchemy-media

Source¶

$ cd /path/to/source
$ pip install .

or

$ pip install -e .

Optional packages¶

Some optional packages are required when using some features, such as: wand and or python-magic.

$ pip install wand python-magic
